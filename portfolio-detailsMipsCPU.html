MIPS CPU

  • Technologies Used: Verilog, Python
  • Company: Imperial College London
  • Project date: 2021
  • Project URL: https://github.com/kert114/Project-MIPS-CPU-

Project Details

  • Led a team of five members in the design and development of a high-performance MIPS-compatible CPU using Verilog
  • Coordinated and integrated individual strengths and ideas within the team to achieve an effective design
  • Focused on creating a production-quality CPU with a robust testing process, aimed at delivering a hardware design expected to work on any FPGA or ASIC
  • Designed and implemented an independent test bench featuring randomized memory stalls, providing a robust mechanism to assert the functional correctness of the CPU
