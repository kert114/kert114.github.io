Augmented Reality Headset

  • Technologies Used:Python, Verilog, React Native, NoSQL
  • Company: Imperial College London
  • Project date: 2022

Project Details

  • Designed and developed a custom headset solution by repurposing a smartphone and constructing a holder to create the AR experience.
  • Developed an Augmented Reality (AR) application using Python, Verilog, React Native and NoSQL
  • Implemented an AR app that provided navigation instructions, real-time weather data, and the ability to locate other users
  • The app could receive commands from a handheld FPGA via TCP communication and access data stored in a DynamoDB database running on AWS
