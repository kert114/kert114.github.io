C-to-MIPS Compiler

  • Technologies Used: C++, Flex, Bison
  • Company: Imperial College London
  • Project date: 2022
  • Project URL: https://github.com/kert114/Compiler-Project

Project Details

  • Collaborated in a pair to develop a preprocessed C90 to MIPS assembly compiler using C++, Flex, and Bison
  • Implemented comprehensive functionality for all basic C constructs, including support for integer and floating-point data types
  • Ensured correctness of the compiler by comparing the output assembly with GCC's compiler using QUMU, an open-source emulator
  • Expanded programming knowledge by working with larger C++ code bases, understanding class structures, and applying object-oriented programming concepts
